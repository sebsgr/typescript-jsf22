Typen

any: Stellt quasi keine Typisierung dar (Kann alles sein), any kann man alles zuweißen

unknown: Stellt das "Gegenteil" zu any dar. Alles kan unknown zugewießen werden, unknown kann aber nur unknown und any zugewießen werden.

never: Bezeichnet einen Wert, der nicht vorkommen kann.

void: Bezeichnet einen Wert ohne Daten. Kann nur undefined (oder null bei strictNullChecks=false) annehmen.
